# RDX Directory

**Purpose:** Centralized home for audits, reports, plans, and task backlogs for the VSM School Merger.

---

## Structure

- `RDX/reports/` - Completed reports, audits, and summaries.
- `RDX/plans/` - Migration and implementation plans.
- `RDX/todos/` - Active, categorized TODO lists.
- `RDX/TODO.md` - Index of all active TODO lists.
- `RDX/readiness.md` - Readiness checklist + report naming standards.
- `RDX/rdx-migration.md` - Task list for RDX organization.

## How To Use

- Add new reports to `RDX/reports/` using the naming convention in `RDX/readiness.md`.
- Update TODOs only in the categorized files under `RDX/todos/`.
- Keep plans in `RDX/plans/` and reference them from TODOs as needed.
