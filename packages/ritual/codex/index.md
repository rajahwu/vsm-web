# Codex

The Codex is the **doctrinal reference layer** of VSM School.

It explains what things mean.  
It does not tell you what to do next.

---

## Tracks

### Genesis — The Alphabet
Pre-literate visual primitives for reasoning about systems.

→ `/codex/genesis`

---

### Source Code — The Doctrine
Sentence-level system reasoning built from primitives.

→ `/codex/source-code`

---

### Powerhouse — Execution & Dexterity
Cognitive and physical alignment under pressure.

→ `/codex/powerhouse`

---

## Rules

- Codex is read-only
- Codex never starts a session
- Codex never tracks progress
- Codex is closed during Prime and Produce

> If the user is acting, Codex is closed.  
> If the user is reading, time is stopped.
