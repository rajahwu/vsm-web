# The Rectangle

## Definition
The Rectangle represents a **process step that consumes time**.

## Function
Rectangles describe what is done, not who does it.

## Operational Use
- Writing
- Coding
- Reviewing
- Sending
- Building

Every Rectangle must have a clear start and end.

## Failure Mode
Unbounded Rectangles cause burnout and infinite work.
