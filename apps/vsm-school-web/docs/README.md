# VSM School Web Docs

This folder groups app-local documentation that used to live at the repo root.

## Structure

- `docs/reports/`: audits, integration summaries, and migration reports.
- `docs/plans/`: migration plans and refactor plans.
- `docs/checklists/`: operational checklists for feature work.
- `docs/guides/`: onboarding and quickstart guides.
- `docs/notes/`: working notes and naming/conventions docs.
- `docs/todos/`: app-scoped TODO lists.
- `docs/blackout/`: Protocol Blackout design and content references.

## Remaining Top-Level Docs

- `apps/vsm-school-web/readme.md`: app overview.
- `apps/vsm-school-web/docs/E2E_TEST_REPORT.md`: test status.
- `apps/vsm-school-web/test-results/...`: Playwright outputs.
